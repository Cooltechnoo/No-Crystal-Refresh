# NoCrystalBounce

A tiny, client-side Fabric mod that removes the annoying "bobbing" animation when you switch to or hold an End Crystal.

### Features
* **Static Viewmodel:** Disables the bobbing/bounce effect specifically for End Crystals.
* **Purely Client-Side:** Works on any server as it only affects your local rendering.
* **Lightweight:** Only modifies one method; zero impact on your FPS.

### Installation
1. Install **Fabric Loader** for 1.21.1 (or 1.21.x).
2. Drop the `.jar` into your `mods` folder.
3. Done. No configuration needed.
